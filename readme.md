Этот проект реализует простую систему учета товаров с классами `Product`, `Smartphone`, `LawnGrass` и `Category`.

## Функции

- Класс `Product` с атрибутами: название, описание, цена и количество.
- Классы-наследники `Smartphone` и `LawnGrass` с дополнительными атрибутами.
- Класс `Category` с атрибутами: название, описание и список товаров.
- Автоматический подсчет количества категорий и товаров.
- Методы для добавления товаров и получения списка товаров.
- Геттеры и сеттеры для атрибутов.
- Магический метод сложения для класса `Product` и его наследников.
- Ограничения на сложение и добавление товаров разных классов.
- Реализован базовый абстрактный класс `BaseProduct`.
- Реализован класс-миксин для вывода информации при создании объекта.


## Установка

1. Клонируйте этот репозиторий.
2. Установите зависимости с помощью `poetry install`.
